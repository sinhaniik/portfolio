# Portfolio Project

## What this is
A personal portfolio for Nikhil — SDE 1 actively transitioning into DevOps,
targeting roles in USA, Canada, Australia, New Zealand, and Europe.

## Purpose
1. Attract DevOps / SDE roles abroad
2. Tell the Dev → DevOps transition story with honesty and clarity
3. Publish blog posts from an ongoing 12-week DevOps learning roadmap
4. Showcase both full-stack (MERN) and DevOps (Linux, Docker, CI/CD, AWS) skills

## Stack
- React 19
- TypeScript (strict)
- Vite 6
- Tailwind CSS v4
- React Router DOM v7
- Redux Toolkit + React Redux
- MDX (to be added: @mdx-js/rollup) for blog posts

## Hosting
Vercel — auto-deploy from main branch

## Theme
Light mode and dark mode both supported.
Clean, minimal design with generous whitespace.
Coffee color palette — dark maroon + warm beige tones.
Font: DM Sans (Google Fonts)
